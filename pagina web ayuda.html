¿como funciona el menu?

El menú de una página web de una biblioteca tiene como objetivo facilitar la navegación y el acceso a la información y servicios disponibles para los usuarios. ¿para que es el boton de inicio?

El botón de inicio es un elemento de interfaz de usuario que se suele encontrar en la barra de navegación de la mayoría de las páginas web. Puede tener diferentes formas y estilos, pero su función principal es la misma: retornar al usuario a la página principal del sitio.

¿para que es el boton de recursos?

Los recursos de inicio de una página web son archivos y elementos esenciales que se cargan primero cuando un usuario visita tu sitio web. Son el "fundamento" de tu página, lo que permite que el resto del contenido se muestre correctamente.
